## The Budget 

JavaScript application to manage budget income and expenses.

### Installing

1. Clone the repo: `git clone https://github.com/marcel-balint/budget.git`

2. Open `index.html` file in your favorite browser.

3. Explore! 

### Technologies Used

* HTML
* CSS
* JavaScript
